웹표준

HTML은 HTML5 DTD의 명세에 맞게 작성하며, w3c html5 문법에 준하여 표준 문법을 사용한다.

접근성

  • 접근성 준수 : Chrome 확장프로그램 Accessibility Insights for Web 자체 체크 및 K-WAH 4.4 웹접근성 자동평가도구 체크 완료 후 파일 전달 한다.
  • 모바일 애플리케이션 접근성지침 2.0을 따른다.
  • 동적 마크업, 상태 변경등 HTML 명세 만으로 충분하지 않은 부분은 WAI-ARIA(Web Accessibility Initiative - Accessible Rich Internet Applications)를 활용하여 보완한다.

기타

하단 클래스는 기본 선언은 되어 있으나 개별 사용은 지양하며, 부모 클래스를 통해 스타이르 선언 하되 예외 사항에서 사용한다.

공통 클래스
클래스명 내용 기타
.hidden 숨김요소 시각적으로 숨김처리되며, 보조기기를 통해서는 정보제공 되는 대상에 적용
.none 숨김요소 시각, 청각 모두 숨김처리대상에 적용 (UI 동적관련 하여 필요시 사용)
.nodata 데이터 없음 상위 클래스 상속 스타일
.sub_con1 상단 여백 40px
.sub_con2 상단 여백 30px
.sub_con3 상단 여백 20px
.txt_c1 포인트 색상 #FF456B
.txt_c2 포인트 색상 #03A4B7
