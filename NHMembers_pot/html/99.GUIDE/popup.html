팝업 호출 방법

<!-- 호출방법1) : 버튼 클릭 이벤트 -->
<button type="button" class="btn_layer btn_close " onclick="openlayer('레이어팝업 id')" >팝업호출</button>

호출하는 버튼에 btn_layer 필수 적용 : 팝업 닫힌 후 초점이동 가능

팝업 닫히는 버튼에 btn_close 적용. 단, 중첩팝업을 띄우는 경우 해당 버튼에 btn_close 사용 금지.(하단 샘플 중 중첩팝업포함 참고)

/* 호출방법2) : 함수 호출 */
openlayer('레이어팝업 id')

팝업 마크업 위치

<!-- 바닥페이지 -->
<div id="wrap"></div>

<!-- 레이어팝업 -->
<div class="layer_pop type_**" id="팝업ID" role="dialog" aria-modal="true" aria-labelledby="타이틀ID"><div>

팝업은 div id="wrap" 밖에 위치한다.
